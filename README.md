<h1 align="center">
  🐕‍🦺 PetCRM** 
</h1>
  <p align="center">Backend for a CRM for pet daycares. Built with NestJS, GraphQL, MongoDB.</p>
  <p align="center">**(need cute name)</p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
